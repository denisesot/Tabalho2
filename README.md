# Tabalho2
Projeto real de um refrigerador com pastilha peltier e sensor de temperatura.

## Materiais

- Caixa de isopor 6L;
- 2 dissipadores;
- 1 pastilha peltier;
- Arduino uno;
- Relé 5V;
- Display Oled;
- Sensor de temperatura - Módulo P10;
- 2 fans;
- 1 fonte ATX;
- 1 placa mãe;
- Cabos;

------------

O código se encontra no arquivo controler.ino disponível neste repositório.
